# 📚 刷题宝 - 题库练习APP

一个纯前端刷题应用，支持多种题型，自带错题本和答题统计。

## 使用方法

### 方式一：直接打开（最简单）
双击 `index.html` 即可在浏览器中使用。

### 方式二：本地服务器（推荐，PWA可用）
```bash
cd quiz-app
python -m http.server 8000
```
然后浏览器打开 `http://localhost:8000`

### 方式三：手机使用
1. 用方式二启动服务器
2. 手机和电脑在同一WiFi
3. 手机浏览器访问 `http://你的电脑IP:8000`
4. 可添加到手机桌面（PWA安装）

## 功能

- 🎯 **刷题模式**：按分类选题，随机打乱顺序
- 📝 **四种题型**：单选题、多选题、判断题、填空题
- ⏱ **计时器**：记录答题用时
- 📊 **统计**：正确率、打卡记录、各分类掌握情况
- 📕 **错题本**：自动收录错题，支持错题重练
- 📥 **导入题库**：支持JSON文件导入自定义题库
- 📱 **PWA**：可安装到手机桌面，离线使用

## 题库JSON格式

```json
{
  "name": "题库名称",
  "version": "1.0",
  "categories": [
    {
      "name": "分类名称",
      "questions": [
        {
          "id": "唯一ID",
          "type": "single",
          "question": "题目内容",
          "options": ["选项A", "选项B", "选项C", "选项D"],
          "answer": [2],
          "explanation": "解析说明（可选）"
        }
      ]
    }
  ]
}
```

### 题型 type 说明
- `single` - 单选题
- `multi` - 多选题
- `tf` - 判断题
- `fill` - 填空题

### answer 格式
- 单选/判断：`[正确选项索引]` 如 `[2]`
- 多选：`[索引1, 索引2, ...]` 如 `[0, 1, 3]`
- 填空：`["正确答案1", "正确答案2"]` 如 `["const"]`

## 数据存储

所有数据存储在浏览器的 localStorage 中，包括：
- 答题进度
- 错题记录
- 统计数据
- 自定义题库

清除浏览器数据会丢失这些记录。
